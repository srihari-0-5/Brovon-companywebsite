# BROVON Company Portfolio

> A sleek, modern single-page portfolio website for the creative agency BROVON.

This project is a fully responsive single-page application built with React, Vite, and Tailwind CSS. It features a modern dark-mode design, smooth scrolling, and subtle animations to showcase the company's services and story.

## ‚ú® Features

* **Modern SPA:** A single-page application for a seamless user experience.
* **Smooth Scrolling:** Implemented with GSAP (`ScrollToPlugin`) for a high-performance, fluid navigation.
* **Component Animations:** Subtle fade-in and hover effects using `Framer Motion`.
* **Responsive Design:** Fully responsive layout for desktop, tablet, and mobile devices.
* **Key Sections:**
    * **Home:** Dynamic hero section.
    * **About Us:** Company story and statistics.
    * **Services:** Interactive grid of company services.
    * **Contact:** Clean contact information section.
